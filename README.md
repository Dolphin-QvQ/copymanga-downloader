# CopyManga 漫画下载器使用指南
## 一、工具介绍
基于 Python + Selenium 实现，可自动抓取漫画章节列表，并批量下载对应章节的图片，支持章节范围选择、反反爬配置、断点续传（基于文件存在性）、详细日志记录等功能。

## 二、环境准备
### 1. 依赖安装
确保已安装 Python 环境（推荐 3.8+），执行以下命令安装依赖包：
```bash
pip install loguru selenium requests
```

### 2. Edge 驱动配置
- 下载与本地 Edge 浏览器版本匹配的 `msedgedriver.exe`（下载地址：[Microsoft Edge Driver](https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/)）
- 将驱动路径配置到代码中 `DRIVER_PATH` 变量（示例：`r"C:\Users\XXX\msedgedriver.exe"`）

## 三、配置说明
修改代码中以下核心配置项（位于代码开头「全局配置」区域）：

| 配置项          | 说明                                                                 |
|-----------------|----------------------------------------------------------------------|
| `BASE_URL`      | 漫画主页 URL（需替换为目标漫画的详情页地址）|
| `SAVE_ROOT`     | 漫画保存根目录（建议使用绝对路径，示例：`r"D:\manga\我的漫画"`）|
| `DRIVER_PATH`   | Edge 驱动程序的绝对路径                                             |
| `MIN_DELAY/MAX_DELAY` | 随机延时范围（单位：秒），防反爬用，建议保留默认值                |
| `PAGE_WAIT`     | 页面加载超时时间（单位：秒），默认 60 秒                             |

## 四、使用步骤
### 1. 启动程序
执行 Python 脚本：
```bash
python copymanga-downloader.py
```

### 2. 页面初始化
- 程序会自动启动 Edge 浏览器并打开配置的漫画主页
- 若页面需要登录/人机验证，需手动完成后等待 5 秒（程序内置等待时间）

### 3. 选择下载章节
程序会列出第一页的所有章节，控制台提示输入下载范围，支持以下格式：
| 输入示例       | 说明                     |
|----------------|--------------------------|
| 直接回车       | 下载所有章节             |
| `2`            | 仅下载第 2 章            |
| `1-3,5,7-10`   | 下载 1-3 章、第 5 章、7-10 章 |

### 4. 自动下载
- 程序会自动进入选中章节，平滑滚动加载所有图片
- 图片按章节创建文件夹，命名规则为「章节标题」，图片按顺序命名（如 `001.webp`、`002.jpg`）
- 下载过程日志会实时输出，同时保存到本地 `manga_download_log_时间.log` 文件

## 五、注意事项
### 1. 路径与命名
- 章节名称中的非法字符（`\/:*?"<>|`）会自动替换为下划线，避免路径错误
- 保存目录不存在时会自动创建

### 2. 防反爬机制
- 内置随机延时、浏览器反检测配置（隐藏 webdriver 特征）
- 图片下载失败会自动重试 3 次，建议不要修改默认延时参数

### 3. 异常处理
- 若浏览器初始化失败，检查驱动路径和 Edge 版本是否匹配
- 若章节列表抓取失败，检查 `XPATH_CHAPTER_ITEM` 等 XPATH 配置是否适配目标网站（网站结构变更可能导致 XPATH 失效）
- 日志文件会记录详细错误信息，可通过日志定位问题

### 4. 其他
- 代码修改周期较长，可能存在遗留bug

## 六、常见问题
### Q1：驱动启动失败？
A1：确认驱动版本与 Edge 浏览器版本一致，驱动路径为绝对路径且无中文/空格。

### Q2：图片下载为空？
A2：检查目标网站图片的 `data-src`/`data-original` 属性是否存在（代码依赖这两个属性解析图片地址），或网站是否有反爬限制（可尝试增加延时）。

### Q3：章节范围输入提示格式错误？
A3：确保输入格式符合规范，例如：
- 正确：`1-5,8`、`10`（英文逗号）
- 错误：`1-`、`a1`、`100`（超出章节总数）
